Технологии Tesla

Tesla использует передовые технологии для создания лучших электромобилей.

Ключевые технологии:

  • Электрические двигатели с высоким КПД
  • Литий-ионные аккумуляторы большой емкости
  • Система быстрой зарядки Supercharger
  • Автопилот для помощи водителю
  • Программное обеспечение с регулярными обновлениями

Все технологии Tesla разрабатываются для повышения эффективности, безопасности и комфорта.
